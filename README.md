# PyChat - 파이챗
파이썬으로 만든 간단한 채팅 프로그램입니다.

평소에 디스코드로 온라인 친구들과 대화를 많이 하는 편인 저는

언젠가 제 손으로 직접 채팅프로그램 하나를 만들고 운영하는게 꿈이였습니다.

아무도 안쓰더라도 적자를 내가면서 소소하게 운영하고 싶었습니다.

그 꿈에 한 발자국 다가간 것 같습니다.

## 프로그램 설명
Google Firebase Realtime DB를 활용해 클라우드 DB에 메시지를 실시간으로 저장, 불러오고

유저 정보를 등록, 방을 등록 하는 등의 기능을 구현하였고

PyQT5로 QT를 활용하여 GUI화면을 구축하였습니다.

## 사용한 PyPI 라이브러리
firebase-admin - 리얼타임 DB 연결 위해 사용

PyQT5 - gui 환경 구축 위해 사용
